
### 一、常用标签

#### 1、video标签

|  **属性**    | **作用**     |
| ------------- |-------------|
| **autoplay** | 是否自动播放 |
| **loop** | 是否循环播放 |

**示例：**

```html
<video muted controls="controls" autoplay="autoplay" loop="loop">
	<source src="video/1.webm" type="video/webm"/>
</video>
```

#### 2、audio标签

|  **属性**    | **作用**     |
| ------------- |-------------|
| **autoplay** | 是否自动播放 |
| **loop** | 是否循环播放 |
| **src** | 文件路径 |

```html
<audio src="music/1.mp3" autoplay="autoplay" loop="loop"></audio>
```
